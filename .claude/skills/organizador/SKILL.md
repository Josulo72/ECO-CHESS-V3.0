# /organizador - Task Organizer & Progress Tracker

## Description
Organizes the current work into clear tasks, tracks progress, and ensures nothing is missed.

## Instructions
When invoked:

1. Read the current `index.html` and identify ALL pending issues
2. Create a numbered task list with priorities (CRITICAL, HIGH, LOW)
3. For each task, specify:
   - What file and line number is affected
   - What the current (broken) code does
   - What the expected behavior should be
4. Track which tasks are done and which remain
5. After all tasks are complete, run a final checklist:
   - [ ] AI works in all 5 levels without Josulo messages (unless Josulo mode)
   - [ ] 3D board renders correctly on mobile (375px width)
   - [ ] 3D board renders correctly on desktop (1200px width)
   - [ ] Login screen, love messages, ECO logo all present
   - [ ] All changes committed and pushed to `main`
